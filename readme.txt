== Description ==

Email Obfuscator will obfuscate email addresses and mailto links in your content using ROT13.

Write email addresses and mailto links as usual, the plugin will automatically find and encode the email addresses.
ROT13 decoding is performed by JavaScript. When JavaScript is disabled or not available, a noscript fallback will 
display the email address as plain text in the form of: name AT domain DOT tld

== Installation ==

= Install =

1. Unzip the `email-obfuscator.zip` file.
1. Upload the `email-obfuscator` folder to your `wp-contents/plugins` folder.

= Activate =

1. In your WordPress admin, go to the Plugins page
1. Activate the Email Obfuscator plugin

If you find any bugs or have any ideas, please email us.
